# Biensperience Design System

Welcome to the Biensperience Design System documentation! This living style guide showcases the components, patterns, and guidelines used throughout the Biensperience travel planning platform.

## Overview

Biensperience is a visual travel experience platform that enables users to plan experiences and share their plans with others. Our design system emphasizes:

- **Visual Storytelling**: Image-rich experiences and destinations
- **Clarity & Guidance**: Clear labels, helper text, and user feedback
- **Consistency**: Unified patterns across all views
- **Accessibility**: WCAG 2.1 compliant components
- **Responsive Design**: Mobile-first approach

## Technology Stack

- **Frontend Framework**: React 18
- **Styling**: Bootstrap 5 + Custom CSS
- **Icons**: React Icons
- **Forms**: Controlled components with validation
- **Routing**: React Router v6
- **Backend**: Node.js + Express + MongoDB

## Design Principles

### 1. User-Centric Design
Every component is designed with the user's needs in mind, providing clear feedback and guidance throughout their journey.

### 2. Visual Hierarchy
Strong visual hierarchy helps users navigate and understand content quickly through:
- Consistent typography scale
- Strategic use of color
- Proper spacing and grouping

### 3. Progressive Disclosure
Show users what they need, when they need it. Complex functionality is broken down into manageable steps.

### 4. Responsive & Accessible
All components work across devices and are accessible to users with disabilities.

## Typography

### Font Family
- **Primary**: Inter, -apple-system, system-ui, sans-serif

### Type Scale
- **H1**: 2.5rem (40px) - Page titles
- **H2**: 2rem (32px) - Section headings
- **H3**: 1.75rem (28px) - Subsection headings
- **H4**: 1.5rem (24px) - Card titles
- **H5**: 1.25rem (20px) - Small headings
- **Body**: 1rem (16px) - Default text
- **Small**: 0.875rem (14px) - Helper text

### Font Weights
- **Regular**: 400
- **Medium**: 500
- **Semi-Bold**: 600
- **Bold**: 700

## Spacing System

Uses Bootstrap's spacing scale (0.25rem increments):
- **xs**: 0.25rem (4px)
- **sm**: 0.5rem (8px)
- **md**: 1rem (16px)
- **lg**: 1.5rem (24px)
- **xl**: 3rem (48px)

Consistent spacing classes:
- `mb-4`: 1.5rem bottom margin (standard between form sections)
- `mt-4`: 1.5rem top margin
- `gap-2`: 0.5rem gap in flexbox
- `p-2`: 0.5rem padding

## Contributing

When adding new components to this design system:

1. **Create a story file**: `ComponentName.stories.js`
2. **Document props**: Use JSDoc comments
3. **Add examples**: Show common use cases
4. **Include accessibility**: ARIA labels and semantic HTML
5. **Test responsive**: Verify mobile, tablet, desktop
6. **Follow patterns**: Match existing component structure
