# Design Tokens

Design tokens are the visual design atoms of the design system — specifically, they are named entities that store visual design attributes. We use them in place of hard-coded values to maintain a consistent and scalable design system.

## Spacing

Bootstrap's spacing utilities use a consistent scale:

| Token | Value | Example Use |
|-------|-------|-------------|
| `0` | 0 | No spacing |
| `1` | 0.25rem (4px) | Tight spacing |
| `2` | 0.5rem (8px) | Small gaps |
| `3` | 1rem (16px) | Default spacing |
| `4` | 1.5rem (24px) | Section spacing |
| `5` | 3rem (48px) | Large sections |

### Common Spacing Patterns

- **Form sections**: `mb-4` (1.5rem bottom margin)
- **Button groups**: `gap-2` (0.5rem gap)
- **Card padding**: `p-3` or `p-4`
- **Page margins**: `my-4` (1.5rem vertical)

## Border Radius

| Token | Value | Usage |

## Animations

### Timing Functions

- **Ease**: `cubic-bezier(0.25, 0.1, 0.25, 1)` - Default smooth
- **Ease In**: `cubic-bezier(0.42, 0, 1, 1)` - Starts slow
- **Ease Out**: `cubic-bezier(0, 0, 0.58, 1)` - Ends slow
- **Linear**: `cubic-bezier(0, 0, 1, 1)` - Constant speed

### Duration

- **Fast**: 0.2s - Micro-interactions
- **Medium**: 0.3s - Standard transitions
- **Slow**: 0.5s - Complex animations

### Common Animations

```css
/* Fade In */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* Fade In Up */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* Shimmer (Loading) */
@keyframes shimmer {
  0% { background-position: -468px 0; }
  100% { background-position: 468px 0; }
}
```

## Usage Guidelines

### Consistency

- Always use design tokens instead of hard-coded values
- Reference tokens through CSS variables or utility classes
- Maintain consistent spacing between similar elements

### Accessibility

- Ensure color contrast ratios meet WCAG 2.1 AA standards (4.5:1 for text)
- Use semantic colors meaningfully
- Don't rely on color alone to convey information

### Performance

- Prefer CSS custom properties for dynamic values
- Use transform and opacity for animations (GPU accelerated)
- Keep animation durations between 200-500ms

## Implementation

### CSS Variables

```css
:root {
  --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  --primary-blue: #667eea;
  --primary-purple: #764ba2;
  --text-primary: #333;
  --text-muted: #6c757d;
  --bg-light: #f7f8fa;
  --border-radius: 1rem;
  --shadow-sm: 0 2px 8px rgba(0,0,0,0.1);
}
```

### Bootstrap Utilities

```jsx

  <h5 className="text-primary fw-bold">Title</h5>
  <p className="text-muted mb-0">Description</p>
```
